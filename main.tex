\section*{1. Analytická časť}

\subsection*{(a) Účelová funkcia}
\[
\min \left\{ J(x) =
-\sum_{i=1}^{m} v^i \ln \left( g(x^T u^i) \right)
+ (1 - v^i) \ln \left( 1 - g(x^T u^i) \right)
\;\middle|\; x \in \mathbb{R}^5 \right\}
\]
Dosadíme sigmoid:
\[
g(z) = \frac{1}{1 + e^{-z}}
\]

\[
\ln \left( g(x^T u^i) \right)
= \ln \left( \frac{1}{1 + e^{-x^T u^i}} \right)
= -\ln \left( 1 + e^{-x^T u^i} \right)
\]

\[
\ln \left( 1 - g(x^T u^i) \right)
= \ln \left( \frac{e^{-x^T u^i}}{1 + e^{-x^T u^i}} \right)
= \ln \left( e^{-x^T u^i} \right)
- \ln \left( 1 + e^{-x^T u^i} \right)
\]

Dosadíme do sumy:
\[
v^i \left(-\ln(1 + e^{-x^T u^i})\right)
+ (1 - v^i)\left( -x^T u^i - \ln(1 + e^{-x^T u^i}) \right)
\]

Uprava:
\[
(1 - v^i)x^T u^i + \ln(1 + e^{-x^T u^i})
\]

Výsledná funkcia:
\[
\min \left\{ J(x) =
\sum_{i=1}^{m} \left[(1 - v^i)x^T u^i + \ln(1 + e^{-x^T u^i})\right]
\;\middle|\; x \in \mathbb{R}^5 \right\}
\]

\subsection*{(c) Gradientná metóda s konštantným krokom}

Metóda bola testovaná pre rôzne hodnoty konštantného kroku $c$
\subsubsection*{$c = 3 \times 10^{-7}$}

\begin{itemize}
    \item \textbf{Výsledok:} $x_c \approx (-0.99398804,\; 3.0594658,\; 4.61121505,\; 5.98086102,\; 0.007168)$
    \item \textbf{Chyba:} $\|x_c - x_{\text{opt}}\|_2 \approx 12.990851628224819$
    \item \textbf{Počet iterácií:} $5\,000\,000$ (limit)
    \item \textbf{Čas:} $137.47\,\mathrm{s}$
\end{itemize}

Krok je príliš malý, metóda konverguje extrémne pomaly a nedosiahne riešenie ani po maximálnom počte iterácií.

\subsubsection*{$c = 3 \times 10^{-5}$}

\begin{itemize}
    \item \textbf{Výsledok:} $x_c \approx (-1.47360963,\; 3.07175527,\; 10.1989855,\; 17.6905539,\; 0.00484668)$
    \item \textbf{Chyba:} $\|x_c - x_{\text{opt}}\|_2 \approx 0.01472689$
    \item \textbf{Počet iterácií:} $2\,736\,859$
    \item \textbf{Čas:} $71.24\,\mathrm{s}$
\end{itemize}

Táto hodnota kroku vedie k stabilnej konvergencii, avšak za cenu veľkého počtu iterácií.

Metóda je stabilná pre c=$3 \times 10^{-5}$, ale pomalá. Počet iterácií je veľmi vysoký, čo je dôsledok pevného kroku bez adaptácie.

\subsubsection*{$c = 3 \times 10^{-3}$}

\begin{itemize}
    \item \textbf{Výsledok:} $x_c \approx (-7.89528871,\; 8.27025728,\; 22.63379642,\; 48.20722059,\; 5.6435618)$
    \item \textbf{Chyba:} $\|x_c - x_{\text{opt}}\|_2 \approx 34.42982206475787$
    \item \textbf{Počet iterácií:} $5\,000\,000$ (limit)
    \item \textbf{Čas:} $134.27\,\mathrm{s}$
\end{itemize}

Krok je príliš veľký, metóda je nestabilná a cieľová funkcia nerastie monotónne, čo vedie k divergencii.

\subsubsection*{Zhrnutie}

Voľba konštantného kroku má zásadný vplyv na konvergenciu:
\begin{itemize}
    \item príliš malý krok spôsobuje veľmi pomalú konvergenciu,
    \item vhodne zvolený krok vedie k stabilnému riešeniu,
    \item príliš veľký krok spôsobuje divergenciu.
\end{itemize}

\subsection*{(d) Gradientná metóda s približne optimálnou dĺžkou kroku}

\begin{itemize}
    \item \textbf{Výsledok:} $x_d \approx (-1.47361102,\; 3.07175538,\; 10.1989558,\; 17.6906325,\; 0.00484666)$

\subsubsection*{Interval \([\lambda_{\min}, \lambda_{\max}]\)}

\begin{itemize}

    \item \textbf{Chyba:} $\|x_e' - x_{\text{opt}}\|_2 \approx 0.7719795862517741$
    \item \textbf{Počet iterácií:} $5\,000\,000$ (limit)
    \item \textbf{Čas:} $282.40\,\mathrm{s}$
\end{itemize}

\subsection*{Zhrnutie}

Z porovnania metód vyplýva:

\begin{itemize}
    \item Konštantný krok je najpomalší a vyžaduje najviac iterácií.
    \item Backtracking výrazne zlepšuje konvergenciu a stabilitu výberu kroku.
    \item Metóda zlatého rezu je citlivá na voľbu intervalu:
    \begin{itemize}
        \item širší interval vedie k lepšej presnosti a rýchlejšej konvergencii,
        \item úzky interval môže spôsobiť stagnáciu a výrazne horšie výsledky.
    \end{itemize}
\end{itemize}

Experimenty potvrdzujú, že efektivita gradientných metód silne závisí od vhodnej voľby dĺžky kroku a škálovania problému.
